# next-solutions
Next Solutions Task


Write a Python application that supports and visualizes a simple database of vehicles. The application must support the following features:

    1. Add a car
Collection of the following data:
    • Bulgarian registration number
    • Brand
    • Model
    • Type of fuel
    • Engine capacity
    • Date of first registration

The registration number must be unique.

    2. Deletion of a car
Deletion of a car from the base according to the submitted registration number

    3. List
Making a list of cars in the database

    4. Search
Getting the cars from the database which in any field contain submitted text


Requirements --> requirements.txt file in the repository
Database - Postgresql
